# 解释
less-loader, ejs-loader, html-loader 的使用

file-loader
* 可以把图片打包到dist目录中
* 可以读取到相对路径的图片

url-loader
* 可以把图片、文件等变成一个base64编码，注入到js文件中，并打包进html中。
* 对小图片，可以这么做

image-webpack-loader
* 压缩图片

# 命令
```
npm i
npm run webpack
```
